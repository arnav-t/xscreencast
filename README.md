# xscreencast
## Screencasting tool for X Window System    
### Usage    
#### Run without any downscaling (slower)    
`xscreencast [port]`     
`xscreencast 4567`          
#### Run with downscaling (faster)    
`xscreencast -d [scale] [port]`     
`xscreencast -d 4 4567`           
Note that the scale must be an integer.
